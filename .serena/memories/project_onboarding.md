# Ocean Guard Project Onboarding

## Purpose
Ocean Guard is a web application designed for a competition, focusing on ocean conservation and deep-sea exploration. It includes an immersive 3D simulation of diving into the ocean.

## Tech Stack
- **Frontend**: React 19, Vite
- **3D Rendering**: React Three Fiber (R3F), @react-three/drei, Three.js
- **Animations**: motion/react (Framer Motion), GSAP, Lottie
- **Styling**: Tailwind CSS, Vanilla CSS
- **Routing**: React Router DOM v7

## Code Style
- Component-based architecture.
- Heavy use of R3F for 3D interactions.
- Modern React patterns (hooks).

## Suggested Commands
- `npm run dev`: Start development server.
- `npm run build`: Build for production.
- `npm run lint`: Lint the codebase.
